# SmartRich Technologies Website

Professional marketing website for SmartRich Technologies - an integrated wealth management platform for Indian Wealth Managers, Distributors, and High-Net-Worth Families.

## Tech Stack

- React + TypeScript
- Express.js backend
- PostgreSQL with Drizzle ORM
- TailwindCSS
- Google reCAPTCHA v3
